📌 如何获取令牌？
1. 访问 GitHub Token 生成页
2. 勾选 gist 权限，滚动到底部点击 "Generate token"
3. 复制生成的令牌（只显示一次），粘贴到上方输入框
4. Gist ID 留空会自动创建，点击激活即可开始使用
💡 令牌仅保存在你的浏览器中，用于在 GitHub 上存储数据
📤 拖拽图片到此处，或点击选择
支持 JPG/PNG/GIF/WEBP，最大 32MB，自动上传到免费图床
